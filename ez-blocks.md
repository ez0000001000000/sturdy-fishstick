# EZ Blocks - Premium Component Library

## Project Overview
A **beautiful, modern component library** built with Next.js 15, React 19, and Tailwind CSS 4. This library provides production-ready, accessible, and customizable UI components for modern web applications.

## Core Features
- **50+ Premium Components** - From basic buttons to complex data visualizations
- **Dark/Light Theme Support** - Built-in theme switching with `next-themes`
- **Fully Responsive** - Mobile-first design approach
- **Accessibility First** - WCAG 2.1 compliant components
- **TypeScript Native** - Full type safety and IntelliSense
- **Animation Ready** - Framer Motion integration for smooth interactions

## Component Categories

### 1. **Layout Components**
- Hero sections with 3D animations (React Three Fiber)
- Grid systems and masonry layouts
- Responsive navigation with mobile menus
- Footer layouts with newsletter signup

### 2. **Form Components**
- Advanced form validation (React Hook Form + Zod)
- Input fields with OTP support
- File upload with drag & drop
- Date pickers and range selectors

### 3. **Data Display**
- Interactive charts and graphs (Recharts)
- Data tables with sorting/filtering
- Progress indicators and loaders
- Syntax highlighters for code

### 4. **Interactive Elements**
- Accordions and collapsibles
- Tabs and step wizards
- Modals and dialogs
- Tooltips and popovers

### 5. **Advanced Features**
- 3D globe visualizations
- Particle effects and animations
- Marquee and carousel components
- Interactive maps and visualizations

## Tech Stack
- **Framework**: Next.js 15 with Turbopack
- **UI**: React 19 + Tailwind CSS 4
- **Components**: Radix UI primitives + shadcn/ui patterns
- **Animations**: Framer Motion + React Three Fiber
- **Forms**: React Hook Form + Zod validation
- **Database**: Drizzle ORM + LibSQL
- **Auth**: Better Auth
- **Styling**: Tailwind with custom animations

## Design System
- **Color Palette**: Modern gradient schemes
- **Typography**: System fonts with fallbacks
- **Spacing**: Consistent 8-point grid system
- **Shadows**: Layered depth system
- **Animations**: Smooth micro-interactions
